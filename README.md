## 🚀 Overview

**Solar System Encyclopedia** is an interactive Unity application that teaches users about the planets in our Solar System.

The application includes a main menu, a full solar system hub, selectable planets, planet information screens, audio narration, mute/unmute controls, animated planets, and custom-made 3D elements.

This project was created for the **COS 742 Unity Class Test 2026** and follows the required test specification, including the main menu, planet hub, planet selection screen, information display, audio features, animated planets, Earth’s moon, and a space shuttle with a particle smoke trail.

---

## ✨ Features

* 🌌 Main menu screen with application title and background music
* ▶️ Start button that opens the main hub
* 🪐 Main hub showing the Solar System planets in order
* 🎵 Background music in the main hub
* 🧭 Planet selection system
* 🌍 Full planet view when a planet is selected
* 🔊 Planet audio narration
* 🔇 Mute and unmute buttons for audio
* ℹ️ Info button that displays planet facts
* 🔙 Return button to go back to the main menu
* 🌙 Custom-modelled Earth moon
* 🎨 Moon texturing with diffuse, bump, and specular detail
* 🔄 Planets rigged and animated to spin on their correct axes
* 🌕 Moon orbiting around Earth
* 🚀 Custom space shuttle model
* 💨 Particle system smoke trail attached to the shuttle
* 🗂️ Consistent naming convention across project files
* 🚫 No Unity Asset Store assets used

## 🧩 Application Flow

1. The user launches the application.
2. The main menu appears with the application title and background music.
3. The user clicks **Start**.
4. The application opens the main hub showing the Solar System.
5. The user selects a planet.
6. A full planet screen appears.
7. The planet’s audio begins playing.
8. The user can mute or unmute the audio.
9. The user can press the info button to view facts about the planet.
10. The user can return to the main menu or continue exploring.

## ▶️ How to Run the Project

### Requirements

Before running the project, make sure you have:

* Unity installed
* The correct Unity version or a compatible Unity version
* Git installed, if cloning from GitHub

---

### Running in Unity

1. Clone or download the repository.

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Open **Unity Hub**.

3. Click **Open**.

4. Select the project folder.

5. Wait for Unity to load and import all assets.

6. Open the main scene from the `Assets/Scenes` folder.

7. Press the **Play** button in Unity.

8. Use the menu and buttons to explore the Solar System application.

## 🎮 Controls

The application is mainly controlled using UI buttons.

| Action            | Control                    |
| ----------------- | -------------------------- |
| Start application | Click **Start**            |
| Select planet     | Click planet button        |
| View planet info  | Click **Info** button      |
| Mute audio        | Click **Mute** button      |
| Unmute audio      | Click **Unmute** button    |
| Return to menu    | Click **Back/Menu** button |
